# Rhythm changes bridge (dominant chain): V/vi → V/ii → V/V → V7

## Обзор

В джазовом стандарте **Rhythm Changes** (форма на основе I Got Rhythm) очень часто встречается бридж из **цепочки доминант** по кругу квинт:

-   **V/vi → V/ii → V/V → V7**

## Как читать обозначения

-   `V/vi` — **вторичная доминанта**: доминанта (V7) по отношению к аккорду `vi`.
-   `V/ii` — доминанта по отношению к аккорду `ii`.
-   `V/V` — доминанта по отношению к доминанте (то есть доминанта доминанты).
-   `V7` — главная доминанта в тональности (доминантсептаккорд).

_Примечание: иногда ту же цепочку записывают как III7 → VI7 → II7 → V7. В ярлыках здесь используется вариант через слэш: `V/vi → V/ii → V/V → V7`._

В C-мажоре это:

-   **E7 → A7 → D7 → G7**

## Зачем это важно

-   цепочки доминант — основа джазового движения и гармонического «ускорения»
-   это отличный тренажёр для ощущения разрешений доминанты и подготовки к ii–V–I

## Практика

1. Играйте цепочку доминант в разных тональностях.
2. Старайтесь минимизировать движение голосов (особенно 3 и 7).
3. После V7 попробуйте разрешать в Imaj7.
