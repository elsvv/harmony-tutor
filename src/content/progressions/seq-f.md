# T64 → S6 → II43 → VII2 → D7 → T3

## Обзор прогрессии

Самая **сложная и драматичная** из базовых прогрессий. Начинается с неустойчивого квартсекстаккорда, проходит через все субдоминантовые и доминантовые обращения, завершаясь неполным трезвучием.

## Структура и логика

| Шаг | Аккорд | Функция           | Бас        |
| --- | ------ | ----------------- | ---------- |
| 1   | T64    | Тоника            | V ступень  |
| 2   | S6     | Субдоминанта      | VI ступень |
| 3   | II43   | Субдоминанта-harm | VI ступень |
| 4   | VII2   | Доминанта-harm    | VI ступень |
| 5   | D7     | Доминанта         | V ступень  |
| 6   | T3     | Тоника (неполное) | I ступень  |

## Уникальные особенности

### VII2 — секундаккорд вводного

**Третье обращение** уменьшённого септаккорда:

-   Септима в басу = **VI♭ ступень** (гармонический мажор)
-   Самое **острое** звучание из всех обращений
-   Максимальное тяготение к разрешению

### Три аккорда на VI ступени

S6, II43 и VII2 образуют **кластер на VI ступени**:

```
VI♮ (S6) → VI♭ (II43) → VI♭ (VII2)
```

Понижение VI ступени происходит **внутри** прогрессии, создавая хроматический сдвиг.

### Драматическая арка

```
Начало         Середина           Конец
  T64 ────────── VI♭ ─────────── T3
   ↓              ↓               ↓
Нестабильно   Максимум       Открытое
              напряжения     завершение
```

## Почему T3 в конце?

После всего драматизма прогрессии неполное трезвучие:

-   Создаёт ощущение **незавершённости**
-   Музыка словно **"выдохлась"** после кульминации
-   Оставляет пространство для **продолжения**

### Акустический эффект

Отсутствие квинты в T3:

-   "Пустое" звучание после насыщенных септаккордов
-   Контраст **полноты → разреженности**
-   Эффект "открытого горизонта"

## Голосоведение

### II43 → VII2

Общие ступени: II, IV, VI♭
Меняется только I → VII (вводный тон появляется!)

### VII2 → D7

-   Бас: VI♭ → V (нисходящая секунда)
-   Септима VII2 становится квинтой D7
-   Вводный тон сохраняется

### D7 → T3

-   Вводный тон → тоника
-   Септима D7 → терция тоники
-   Квинта D7 "исчезает" (потому T3 без квинты)

## Историческое значение

Подобные обороты характерны для:

-   **Поздний Бетховен** — драматические медленные части
-   **Шуберт** — песни с неразрешённой тоской
-   **Брамс** — сложные гармонические последовательности

## Практические советы

1. **T64**: Играйте мягко — это лишь отправная точка
2. **Хроматический сдвиг** на VI: почувствуйте "потемнение" гармонии
3. **VII2 → D7**: Ключевой момент — не торопитесь
4. **T3**: Дайте звуку растаять в тишине
