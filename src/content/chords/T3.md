# T3 — Неполное тоническое трезвучие

## Структура

**Неполное трезвучие** — только два звука:

-   **Прима** (I ступень) — в басу
-   **Терция** (III ступень)
-   ~~Квинта~~ — отсутствует

## Интервальный состав

-   Большая терция (4 полутона) между I и III
-   Нет квинты!

## Почему без квинты?

### Голосоведение из D7

При разрешении доминантсептаккорда:

-   Вводный тон (VII) → тоника (I)
-   Септима (IV) → терция (III)
-   Квинта (II) → тоника (I)
-   Прима (V) → тоника (I)

Все голоса разрешаются в **приму и терцию**, квинте "неоткуда взяться"!

### Правило классической гармонии

Чтобы получить полное T53 после D7, нужно:

-   Допустить скачок в одном из голосов
-   Или нарушить естественное тяготение

T3 — это **грамотное** разрешение без компромиссов.

## Звучание

-   "Открытое", незавершённое
-   Больше пространства, меньше плотности
-   Эффект "выдоха" после напряжённой доминанты

## Когда появляется

-   После D7 в строгом голосоведении
-   В заключениях с эффектом "растворения"
-   Когда нужна тоника, но не финальная точка

## Сравнение

| T53              | T3          |
| ---------------- | ----------- |
| Полное трезвучие | Неполное    |
| 3 звука          | 2 звука     |
| Плотное звучание | Разреженное |
| Финальность      | Открытость  |

## Практический совет

Не "добавляйте" квинту интуитивно — T3 должен звучать именно так. Это не ошибка, а осознанный выбор гармонии.
