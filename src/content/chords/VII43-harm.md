# VII₄₃ гарм. — Терцквартаккорд VII ступени гармонического мажора

## Структура

**Второе обращение уменьшённого септаккорда** — квинта в басу:

-   **Квинта** (IV ступень) — в басу
-   **Септима** (♭VI ступень — **пониженная!**)
-   **Прима** (VII ступень)
-   **Терция** (II ступень)

## Особенность гармонического мажора

Пониженная VI ступень делает аккорд **полностью уменьшённым** (°7):

-   Уменьшённая септима (вместо малой)
-   Симметричная структура из малых терций
-   Максимальное гармоническое напряжение

## Интервальный состав от баса

-   Терция (м.3)
-   Кварта (тритон / ув.4)
-   Секста (б.6)

## Функция

**Доминантовая функция с IV ступенью в басу**:

-   Субдоминантовый бас + доминантовая функция
-   Создаёт эффект "двойного тяготения"
-   Часто используется перед кадансовым квартсекстаккордом

## Типичное разрешение

В T₆₄ (кадансовый квартсекстаккорд):

```
IV → V (бас идёт вверх на секунду)
♭VI → V (пониженная VI разрешается вниз)
VII → I
II → I или II → III
```

## Сравнение обращений

| Обращение       | Бас    | Характер                |
| --------------- | ------ | ----------------------- |
| VII₇ гарм.      | VII    | Острый, вводнотоновый   |
| VII₆₅ гарм.     | II     | Мягче, II в басу        |
| **VII₄₃ гарм.** | **IV** | Субдоминантовый оттенок |
| VII₂ гарм.      | ♭VI    | Максимально красочный   |

## Историческое применение

-   **Бах**: в хоральных каденциях
-   **Венские классики**: перед K₆₄
-   **Романтики**: колористические эффекты

## Практический совет

IV ступень в басу создаёт **мостик между S и D** — аккорд объединяет обе функции, что делает его идеальным для плавного перехода к каденции.
