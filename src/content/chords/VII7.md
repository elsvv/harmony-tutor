# VII7 — Уменьшённый вводный септаккорд

## Как читать обозначение

Название `VII7` можно разобрать так:

-   `VII` — римская цифра: аккорд строится от VII ступени (вводного тона), поэтому он очень сильно тяготеет к тонике.
-   `7` — септаккорд в основном виде: в басу находится прима аккорда.

## Структура

**Основной вид** септаккорда VII ступени (гарм. мажор):

-   **VII ступень** — в басу (вводный тон!)
-   **II ступень** — терция
-   **IV ступень** — квинта
-   **VI♭ ступень** — септима (гармонический мажор)

## Интервальный состав

-   Малая терция (VII → II)
-   Уменьшённая квинта (VII → IV) — тритон!
-   Уменьшённая септима (VII → VI♭)

## Функция

**Доминанта** максимальной интенсивности:

-   Содержит **два тритона**: VII-IV и II-VI♭
-   Каждый звук имеет сильное тяготение
-   "Двойная доминанта" перед D65

## Два тритона

```
VII ←─тритон─→ IV
II  ←─тритон─→ VI♭
```

Это создаёт **максимальное напряжение** в тональной системе.

## Разрешение

### VII7 → D65 (типичное)

```
VII (бас) → VII (остаётся)
II        → II
IV        → V
VI♭       → V
```

### VII7 → T (прямое)

```
VII → I
II  → I или III
IV  → III
VI♭ → V
```

## Гармонический мажор

Без VI♭ получился бы **полууменьшённый** аккорд (м7).
С VI♭ — **полностью уменьшённый** (ум7), что:

-   Усиливает напряжение
-   Создаёт симметричную структуру
-   Позволяет энгармонические модуляции

## В контексте прогрессии D

```
... → II2 → VII7 → D65 → T53
            ↑
    Кульминация напряжения
```

## Исторический факт

Уменьшённый септаккорд — любимый аккорд **романтиков** (Бетховен, Шуберт, Лист). Его симметрия позволяет модулировать в любую тональность!

## Практический совет

VII7 — точка максимального напряжения. Дайте ему прозвучать, не торопитесь к разрешению.
